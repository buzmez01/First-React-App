# First React Project

A progressive React learning project built with Vite. Each mini-app introduces new React concepts step by step.

## Projects

| # | Project | Concepts Covered |
|---|---------|-----------------|
| 1 | **Todo App** | useState, JSX, event handling, list rendering with map(), immutability |
| 2 | **Weather App** | useEffect, fetch API, async/await, conditional rendering |
| 3 | **Notes App** | *Coming soon* |
| 4 | **Movie List** | *Coming soon* |

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Tech Stack

- React 19
- Vite 8
- JavaScript (no TypeScript — intentional for learning)
- Plain CSS (no frameworks)
